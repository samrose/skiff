# Code of Conduct

Skiff adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text is published at the link above and is incorporated here by reference.

## Scope

This Code of Conduct applies to all skiff project spaces: the GitHub repository, issue tracker, pull requests, discussions, and any direct project communication. It also applies when an individual is officially representing the project in public spaces.

## Our pledge

We pledge to make participation in skiff a respectful and welcoming experience for everyone, regardless of background or identity. Maintainers commit to acting and interacting in ways that contribute to an open, inclusive, and healthy community.

## Enforcement

Concerns about conduct may be reported privately to the project maintainers at `conduct@skiff-build.example` (TODO: replace with a real address before public launch). All reports will be reviewed promptly and handled in accordance with the enforcement guidelines defined in the Contributor Covenant.

Reports will be kept confidential to the extent practical. Maintainers who violate the Code of Conduct are subject to the same enforcement as anyone else.
